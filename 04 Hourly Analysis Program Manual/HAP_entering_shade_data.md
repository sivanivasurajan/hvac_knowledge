Overview for External Shade Geometries and the Shade Form
About External Shades. An external shade geometry defines the dimensions of the reveal, overhang and fins
which externally shade a window. Shade dimensions are used together with solar position data in HAP load
calculations to determine the fraction of the window surface shaded by reveals, overhangs and fins. This in
turn affects the solar heat gain and solar load for the window.
Often shading devices having common dimensions are used with most or all of the windows in a building
facade. A shading geometry can be created in HAP for each of different set of shading dimensions used. Each
shading geometry can then be reused for many different windows in the building facade, thereby saving the
time of having to specify the complete set of dimensions over and over again for each space.
The Shading Geometry Form is used to define the dimensions of a reveal, an overhang, a right-hand fin and
a left-hand fin which externally shade a window. This form contains the twelve input items divided into five
groups:

The form also contains three command buttons in the lower right hand corner:
1. Press OK to save the shade geometry inputs and return to the HAP main program window.
2. Press Cancel to return to the HAP main program window without saving changes to the shade geometry.
If you press Cancel while creating a new shade geometry, a new shade geometry will not be added to
your shade library.
3. Press Help to display this overview discussion of shade geometries and the shade input form.

Shade Name
This is a reference name for the external shade geometry. The shade name will appear on a selection list
when you are adding external shade to a space. The name also appears on reports listing shading inputs. The
name should be descriptive so you can easily recognize what the data represents later when the name
appears on selection lists or reports.

Overhang Dimensions
An Overhang is a horizontal projection from the building surface above a window as shown in the figure below.
To analyze how an overhang shades a window, four dimensions must be defined:
1. Projection From Surface defines the distance between the wall and the outer edge of the overhang
measured perpendicular to the building surface.
2. Height Above Window defines the distance between the top edge of the window aperture and the bottom
edge of the overhang.
3. Extension Past Right-Hand Side of Window, Extension Past Left-Hand Side of Window define how far
the overhang extends beyond the right and left-hand sides of the window aperture. Right and left refer to
directions when viewing the exterior surface of the building and the window from outside.

Fin Dimensions
A Fin is a vertical projection from the building surface to the right and/or left of the window. Right and left refer
to directions when viewing the exterior surface of the building and the window from outside. The figure below
illustrates the geometry of a fin to the right of the window. The dimensions for the right fin and left fin are
defined separately. To calculate how each fin shades a window, three dimensions must be defined:
1. Projection From Surface defines the distance between the building surface and the outer edge of the fin,
measured perpendicular to the building surface.
2. Height Above Window defines the distance between the top edge of the window aperture and the top edge
of the fin, measured parallel to the fin.
3. Distance From Edge of Window defines the distance between the edge of the window aperture and the
edge of the fin.
